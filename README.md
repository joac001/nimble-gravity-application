Esta es mini aplicacion es parte de mi postulacion para el puesto como Jr. Full-stack developer en Nimble Gravity.

Para usar la aplicacion:

Renombrar `.env.example` a `.env`
Completar con la URL de la API y con el mail del candidato

```bash
VITE_BASE_URL=https://url-base.aqui/
EMAIL=candidato@gmail.com
```

Correr los comandos

```bash
pnpm install
pnpm dev
```
